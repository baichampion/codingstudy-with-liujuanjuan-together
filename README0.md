#git的一些基本操作记录

> 克隆clone：git clone url
与云端同步：git pull
向云端提交更改系列命令：1、git add -A(增加全部文件) 2、git commit -m "备注更改情况"（提交更改备注） 3、git push origin master (向主分支推送更改)

#根目录的操作命令
> 切换命令，cd 是切换到根目录的下一个文件夹，cd..是切换到根目录的上一个文件夹。
